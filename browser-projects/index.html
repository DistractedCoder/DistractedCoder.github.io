Checkbox Rendering Engine

Checkbox Digital Clock

I was bored so I made a digital clock using checkboxes. That's all.

Checkbox game Part 1:Bouncing Balls

Taking the idea further, I made an entire checkbox screen. This was the first part of a multi-part project.

Checkbox game Part 2:'Sprite' Support

If it's going to become a game, it needed sprite support. This was just a test to make sure that was all working.

Checkbox game Part 3:Final UI

Checkbox UI. You can import you're own text into it.

Checkbox game Part 4:Snake Game

Snake was the first official game I made on the checkbox screen. Made logical sense since everything was grid based already.

Checkbox game Part 5:New Rendering System

The system wasn't very efficient. I was updating everything every frame even if it was already enabled. I updated it to only update the changed checkboxes and that made a massive difference in performance.

Checkbox game Part 6:3D Objects

Just for fun I made it render some 3D objects. Lot's of matrix math, bleh.

Checkbox game Part 7:Dithering

The screen only supported 2 colors. That's not much.
So I tried to add dithering to simulate a 50% blue color. It sort of worked but the browser cannot stutter for even a second or the thing fails.
It might not even work any more.

Checkbox game Part 8:Analog Clock

I already made a digital clock so logically I made an analog one too.

Checkbox game Part 9:Start of a Platformer

Now I need a better game than snake. I started the basics on a platformer. It has moving and jumping.

Checkbox game Part 10:Actually a Platformer

A full platformer game. It has a player, enemies, and a level.

Checkbox game Part 11:DOOM

It's an implementation of the classic DOOM game using the checkbox system. It's not a port but my own interpretation of the game.
It's still crazy that this was possible to do at all!
